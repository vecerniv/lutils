Quickstart
----------
1. Install this library from PyPI:

.. code-block:: bash

   pip install lutils

2. Or directly from GitHub:

.. code-block:: bash

   pip install git+https://github.com/vecerniv/lutils.git

3. Create your own script based on the reference or check out the examples!
