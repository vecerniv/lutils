# lutils
The Lambda Utilities package was created to analyze the accuracy of the lambda field generated for the [openHFDIBRANS](https://github.com/LucieKubickova/openHFDIBRANS) solver.\
The package also includes tools for logging and plotting the results.

# Documentation
Refer to the [documentation](https://vecerniv.github.io/lutils) for more information.
